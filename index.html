Research

Public Key Searchable Encryption Using Elliptic Curve Diffie-Hellman Cryptography

In this paper, we addressed four important issues of the VPKSE scheme. This scheme allows any person to generate a trapdoor, cipher text can be easily modified, the key pair is generated by the users themselves, and the identities of the users are encrypted leading to unnecessary inefficiency. To this end, a ‘Public key Searchable Encryption Scheme’ is proposed
